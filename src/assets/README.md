# Assets

Эта папка содержит все статические ресурсы проекта (изображения, шрифты и т.д.).

## Структура папок

### 📁 images/
Изображения проекта, организованные по категориям:

- **logo/** - Логотипы и брендинг
  - Основной логотип клуба
  - Иконки для фавиконов
  - Варианты логотипа (светлый/тёмный)

- **icons/** - Иконки и пиктограммы
  - Иконки для UI элементов
  - Иконки социальных сетей
  - Декоративные иконки

- **gallery/** - Фотографии для галереи
  - Фото с турниров
  - Фото атмосферы клуба
  - Фото игроков (с разрешения)

- **tournaments/** - Изображения, связанные с турнирами
  - Баннеры турниров
  - Фото призовых фондов
  - Изображения для карточек турниров

- **backgrounds/** - Фоновые изображения
  - Фоны для секций
  - Декоративные паттерны
  - Градиенты и текстуры

### 📁 fonts/
Кастомные шрифты (если используются локальные файлы шрифтов)

## Использование

### Импорт изображений в компонентах:

```jsx
// Пример импорта логотипа
import logo from '../assets/images/logo/ante-club-logo.svg'

// Пример импорта изображения для галереи
import tournamentPhoto from '../assets/images/gallery/tournament-1.jpg'
```

### Использование в JSX:

```jsx
<img src={logo} alt="Ante Club Logo" />
```

## Рекомендации

- Используйте осмысленные имена файлов (например: `tournament-january-2025.jpg` вместо `img1.jpg`)
- Оптимизируйте изображения перед добавлением (сжимайте, используйте WebP где возможно)
- Для больших изображений используйте lazy loading
- Храните оригиналы в отдельной папке `originals/` если нужно
